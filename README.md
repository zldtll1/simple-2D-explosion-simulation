#具有爆炸效果的简单二维流体模拟
代码主要基于项目：https://github.com/tunabrain/incremental-fluids 。主要想法是将不可压缩流体变为可压缩流体，实现方法是对投影后的流体（已经满足不可压缩性）添加人工压力（变为可控的可压缩流体）。本方法主要是实现视觉效果，并不十分遵循物理原理。
